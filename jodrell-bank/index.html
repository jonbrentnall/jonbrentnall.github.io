Jodrell Bank

This site was inherited from another agency, we were asked to maintain and update the site when required (there wasn't the budget available to update the site). As the codebase is out of date this has lead to some interesting solutions where we've tried to get the existing code to work well alongside anything we add.

jodrellbank.net

Even though we've not been able to rebuild the site we have made considerable improvements in speed and performance, we've managed to increase inner page speed times by removing unused code and optimizing, where possible, the existing code. As well as smaller changes such as simply running the images through a compressor (tinyPNG).

Given the time and budget, it would be great to completely rewrite the css used throughout the site. It uses a very old responsive library where 80% - 90% isn't used, as well as reducing the site's dependency on plugins.
